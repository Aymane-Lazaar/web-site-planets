# Planets Interactive Menu

This is a simple HTML and CSS project that creates an interactive planets menu. Each planet has a hover effect with animations to simulate glowing and scaling effects.

## Project Structure


- `index.html` → Main HTML file containing the planets menu structure.
- `style Planets.css` → CSS file for styling and hover effects.
- `stars-img/stars.png` → Background stars image used in the page.

## Features

- Interactive menu for the Sun and planets in the Solar System.
- Hover effects:
  - Scaling of planet names.
  - Glowing circle effect behind each planet.
  - Large, semi-transparent text overlay with planet name.
- Gradient background with starry sky image.
- Modern, minimal design with centered content.
